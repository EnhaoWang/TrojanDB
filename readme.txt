This project(database) is named TrojanDB.

To launch the program, simply run main.py (python3 main.py). Below are the functionalities of each code file:

main.py: Launches the program and continuously provides a command-line interface.
parse.py: Parses the database language, checks for errors, or extracts information from the database language, called by main.py.
funcs.py: Executes various database functions, called by parse.py.
utils.py: Provides some common variables and functions for the first three files.

testcases.txt contains some test cases used for demo.
